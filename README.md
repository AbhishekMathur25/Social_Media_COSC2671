PROJECT OVERVIEW
================================================================

This project investigates whether YouTube financial influencers
can predict or influence ASX200 returns. The analysis covers:

  - Sentiment analysis of 387 video titles and 15,811 comments
  - Topic modelling (LDA) identifying 5 content domains
  - Network construction using commenter overlap across 53 channels
  - Community detection using the Louvain algorithm
  - Information diffusion tracking across 8 key financial topics
  - Homophily testing across sentiment, topic, and community

Data collection window: January 2024 to December 2025
ASX trading days covered: 507

================================================================
FILE RUN ORDER
================================================================

Scripts must be run in the following order. Each script reads
from the CSV files produced by the previous step.

  Step 1:  01_data_collection.py       <- Collects YouTube + ASX data
  Step 2:  02_preprocessing.py         <- Cleans and filters data
  Step 3:  03_sentiment_analysis.py    <- VADER sentiment on titles/comments
  Step 4:  04_topic_modelling.py       <- LDA topic modelling
  Step 5:  05_network_construction.py  <- Builds commenter overlap network
  Step 6:  06_community_detection.py   <- Louvain community detection
  Step 7:  07_information_diffusion_homophily.py  <- Diffusion + homophily

NOTE: All intermediate CSV files from Steps 1-7 are already
included in this submission. You can skip Step 1 and run
Steps 2-7 directly using the provided data files.
